NooBaa S3 Compatibility

Protocol Features

Supported Protocol Features:

  • Authentication - Sigv4, Sigv2, Presigned URL.
  • Authorization - Bucket Policy.
  • Bucket addressing - Path-style ("domain.io/bucket"), Host-style ("bucket.domain.com").
  • Common Request Headers - Authorization, Content-Length, Content-Type, Content-MD5, Date, Expect, Host, x-amz-content-sha256, x-amz-date, x-amz-security-token.
  • Common Response Headers - Content-Length, Content-Type, Connection, Date, ETag, Server.

Unsupported Protocol Features:

  • Bucket or Object ACL (x-amz-acl and x-amz-grant headers) - use Bucket Policy instead.
